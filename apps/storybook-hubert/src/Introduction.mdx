# Hi ðŸ‘‹. This is Hubert. A slim component library built for lotta

Hubert is a slim component library we use for the
[lotta project](https://lotta.schule), a modern, easy-to use open source
solution for the educational sector which wants to get privacy and security right.

## Install

Install the package via npm

```bash
npm install @lotta-schule/hubert
```

At the top of your react tree, add the Provider, which loads some basic styling:

```tsx
import { HubertProvider } from '@lotta-schule/hubert';

export const MyTopComponent = () => (
  <HubertProvider>
    <MyMainComponent />
  </HubertProvider>
);
```

You can also pass an object with supported fonts to the provider:

```tsx
import { HubertProvider } from '@lotta-schule/hubert';

export const MyTopComponent = () => (
  <HubertProvider
    supportedFonts={[
      {
        name: 'Quicksand',
        url: 'https://fonts.googleapis.com/css2?family=Quicksand:wght@300&display=swap',
      },
    ]}
  >
    <MyMainComponent />
  </HubertProvider>
);
```
